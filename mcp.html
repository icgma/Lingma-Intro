MCP 服务 详解

Model Context Protocol — 让 AI 编码助手连接外部工具和数据源，无限扩展能力边界。

📖 开始阅读 ← 返回设置

什么是 MCP？

MCP（Model Context Protocol，模型上下文协议）是一种开放标准协议，为大语言模型提供了标准化的外部工具和上下文集成方式。

🤔 通俗理解

你可以把 MCP 想象成给 AI 助手安装的"App Store"。每个 MCP 服务就像一个 App，让 AI 具备新的能力——比如查数据库、读文档、调 API 等。

💡

没有 MCP 时，AI 只能读你当前的代码文件。有了 MCP，AI 可以主动去获取外部信息，大大提升工作效率。

🏗️ 技术架构

MCP 服务支持两种通信模式：

1

STDIO 模式

通过标准输入输出流通信，服务在本地运行。适合轻量级工具，启动快。

2

SSE 模式

采用服务器发送事件协议通信，服务可在远端或本地。适合复杂服务或共享服务。

🌐

开放标准

MCP 不是灵码独有的协议，是行业通用标准。学会了在灵码中使用，到其他支持 MCP 的工具中也同样适用。

🧩

即插即用

大多数 MCP 服务开箱即用，只需在设置中添加即可。无需编写代码或修改配置文件。

🤖

智能体自主调用

编程智能体会根据你的需求描述，自主判断是否需要调用 MCP 工具，并在执行前征求你的确认。

如何添加 MCP 服务

两种方式：从 MCP 广场一键安装，或手动配置服务。

方式一：从 MCP 广场安装（推荐）

1

打开设置

点击左下角齿轮图标，或通过命令面板搜索 Lingma: Settings。

2

进入 MCP 服务页面

在左侧导航中点击 MCP 服务，切换到 MCP 广场 标签页。

3

搜索并安装

浏览或搜索所需的 MCP 服务，点击 安装 按钮。安装完成后切换到"我的服务"页面，图标显示为 绿色 即表示连接成功。

方式二：手动添加

1

点击「+」按钮

在 MCP 服务页面右上角，点击 + 号。

2

选择添加方式

可以选择 手工添加 或 导入 JSON 配置。

3

填写服务信息

根据服务类型填写：

{ "name": "my-tool", "command": "npx", "args": ["-y", "@example/mcp-server"], "env": { "API_KEY": "your-key" } }
{ "name": "remote-tool", "url": "https://mcp.example.com/sse", "headers": { "Authorization": "Bearer xxx" } }

MCP 广场

通义灵码集成了魔搭社区 MCP 市场，提供 3800+ 个 MCP 服务，覆盖十大热门领域。

🛠️

开发者工具

GitHub、GitLab、Jira、Linear 等项目管理和代码托管工具集成。

📁

文件系统

本地文件/目录的增删改查操作，支持批量处理。

🗄️

数据库

MySQL、PostgreSQL、MongoDB 等数据库连接，自动生成查询和 DAO 代码。

🔍

搜索引擎

让 AI 联网搜索最新信息，获取实时的技术文档和教程。

📐

设计工具

MasterGo、Figma 等设计稿读取，自动将设计稿转换为前端代码。

🚀

DevOps

云效 DevOps 集成，实现从工作项到代码生成、提交的自动化流程。

📊

数据分析

连接数据分析平台，AI 自动进行数据查询、统计和可视化。

🗺️

地图服务

地图 API 集成，适合需要地理信息处理的应用开发。

🏋️

算法练习

LeetCode MCP 服务，在 IDE 内随机获取算法题并辅助解题。

实际应用场景

以下是几个常见的 MCP + 智能体协作示例，展示 MCP 如何提升开发效率。

📝

场景一：根据在线文档生成代码

配置 Fetch MCP 服务后，可以告诉 AI："参考这个 API 文档链接，帮我写一个 Python 客户端"。AI 会自动读取网页内容并生成对应代码。

# 你对 AI 说： 参考 https://api.example.com/docs ， 帮我写一个获取用户列表的 Python 函数。 # AI 会： # 1. 调用 Fetch MCP 读取网页内容 # 2. 分析 API 文档结构 # 3. 生成完整的 Python 代码
🗄️

场景二：数据库辅助开发

配置数据库 MCP 后，AI 可以直接查看数据库结构，自动生成 SQL 查询语句和数据访问代码。

🎨

场景三：设计稿转代码

配置 MasterGo 或 Figma MCP 后，AI 读取设计稿中的组件信息，自动生成对应的 HTML/CSS 代码，大幅减少前端还原工作量。

🏋️

场景四：在 IDE 内刷算法题

配置 LeetCode MCP 服务后，可以对 AI 说："随机给我一道中等难度的数组题"。AI 调用 MCP 获取题目，你在 IDE 里写代码，AI 辅助调试。

⚠️

注意：部分 MCP 服务需要配置 API Key 或其他凭证。首次使用时请仔细阅读服务提供方的说明文档。智能体在调用 MCP 工具前会先征求你的确认。
